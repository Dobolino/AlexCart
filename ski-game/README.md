# Peak Dual

Browser-Ski-Spiel für PC im Stil von *Grand Mountain Adventure* — Dual-Daumen-Steuerung,
übersetzt auf Tastatur und Maus.

## Spielen

```bash
cd ski-game
npm install
npm run dev
```

Dann im Browser öffnen (meist `http://localhost:5174`).

## Steuerung

| Aktion | PC-Eingabe |
|--------|------------|
| Linker Ski/Stock | `A` oder linke Maustaste |
| Rechter Ski/Stock | `D` oder rechte Maustaste |
| Beschleunigen | Beide gleichzeitig |
| Kurve links | Nur rechts halten |
| Kurve rechts | Nur links halten |
| Scharf bremsen | `Shift` + beide |
| Switch (180°) | `Ctrl` + beide |
| Frontflip / Backflip | `W` / `S` (in der Luft) |
| Spin | `A` oder `D` halten (in der Luft) |
| Grab | `Q` / `E` (in der Luft) |

Ziel: Tore durchfahren, Tricks landen, die Piste bis zum Ende fahren — ohne Bäume und Felsen.
